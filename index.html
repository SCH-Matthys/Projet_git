Projet git

commandes git

  • Initialiser le projet git avec [ git init ]
  • Ajouter la page ou le projet pour le commit [git add index.html ] ou [ git add . ]
  • Commit le projet avec la commande [ git commit -"Nom pertinent du commit"
  • Pour ajouter de nouveeaux fichers/dossier au projet repeter la meme opération [ git add ] [ git commit -m"message" ]
  • Pour savoir voir l'état global du projet [ git log ]
  • Pour voir l'état du commit actuel [ git status ]
  • Utiliser exemple : [ git checkout 5513543543845314545] pour se deplacer sur les ancien commit
  • [ git log ] pour verifier qu'on n'est plus sur la branche master
  • [ git tag "nom_du_tag"] pour ajouter un tag sur un ancien commit
  • [ git checkout master ] pour retourner sur la brancher principale
